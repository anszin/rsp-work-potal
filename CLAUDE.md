##프로젝트명
플랫폼팀 업무 포탈

## 업무 요건
1. 요청 관리(운영시스템의 변경 요청 관리, 현재 운영 시스템 2개, 추후 운영시스템 추가 예정)
2. 배포 관리(운영시스템의 배포 요청 관리, 현재 운영 시스템 2개, 추후 운영시스템 추가 예정)
3. 인벤토리 관리(수주/제안 아이템 관리)
4. 회의록/주간보고 정리, 공유
5. 일일점검 보고(우선 시스템 담당자 수동보고, 향후 자동화 예정)
6. 손익 관리(월별 매출/지출 관리)
7. 종합 대시보드
